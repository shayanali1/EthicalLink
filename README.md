# EthicalLink — Loan & Installment Planner

A web-based financial tool built with Python and Flask that helps users plan loan repayment schedules.

## Features
- Interest-free Islamic loan calculator (Qard al-Hasan)
- Standard fixed installment plan calculator
- Clean, simple web interface

## Tech Stack
- Python 3.13
- Flask 3.1.3
- HTML and CSS

## How to Run Locally
1. Clone the repository
2. Create a virtual environment using `py -m venv venv`
3. Activate it using `venv\Scripts\activate`
4. Install dependencies using `py -m pip install -r requirements.txt`
5. Run the app using `py app.py`
6. Open your browser and go to `http://127.0.0.1:5000`
